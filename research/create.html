📋 디자이너 노트: 연구 생성 Step 1 — AI 인터뷰

🟡 우선순위: Phase 1B (3~6개월)

연구 생성 6단계 중 첫 번째. AI가 대화형으로 연구 아이디어를 구체화하는 인터뷰 화면. 3-panel 레이아웃(사이드바 + 채팅 + 레포트).

레이아웃 구조
  • 사이드바 (240px): 로고 + 네비게이션 + 내 연구 목록(활성 상태 dot: 초록=진행중, 파랑=모집중, 회색=초안) + 임시저장 목록(노란 dot=최근, 회색=오래됨) + 사용자 정보
  • Step Bar (상단): 6단계 진행 표시. 현재=보라, 완료=초록, 미진행=회색. 1.AI인터뷰 → 2.연구설계 → 3.데이터수집 → 4.타겟설정 → 5.IRB판정 → 6.에스크로
  • 채팅 패널 (중앙): AI 대화 영역. 아래에 진행 요약 바 + 입력창
  • 레포트 패널 (우측 400px): 실시간 RAG 검색 결과. PubMed/학술DB에서 관련 논문 자동 표시
채팅 패널 구성
  • AI 메시지 (좌측 정렬): 흰 배경, 라운드 말풍선. 분석 내용 + 후속 질문 + 제안 칩 포함
  • 사용자 메시지 (우측 정렬): 보라 배경, 사용자 답변
  • 제안 칩: AI가 연구 주제 예시를 칩 형태로 제공. 클릭 시 입력란에 자동 채움
  • 진행 요약 바: 7개 항목(주제/대상/개입/기간/결과지표/방법론/가설). 채워진 항목은 초록 dot + 굵은 글씨
레포트 패널 구성
  • 연구 동향: 관련 RCT 수 + RCT 비율 통계 박스. 최근 연구 트렌드 텍스트 요약
  • 관련 논문: 제목 + 저자 + 저널 + DOI + 요약 스니펫. 카드형 리스트
  • 샘플사이즈 근거: 파워 분석 기반 최소 필요 인원 + 탈락률 고려 목표 인원
  • 관련 키워드: 태그 칩으로 표시. 검색에 사용된 키워드들
임시저장 자동 저장 동작
  • 트리거: 사용자가 첫 메시지 전송 시 자동으로 임시저장 생성
  • 제목 자동 생성: AI가 대화 내용에서 연구 주제를 추출하여 임시저장 제목 자동 부여 (예: "건강 데이터 코호트 분석 연구")
  • 저장 주기: 매 메시지 전송 시 + 30초 idle 자동 저장
  • 사이드바 반영: 현재 세션이 임시저장 리스트 최상단에 보라 하이라이트로 표시. pulse dot으로 활성 상태 표현
  • 채팅 헤더: "자동 저장됨" 뱃지가 fade 애니메이션으로 반복 표시
  • Step 진행 시: 다음 Step으로 넘어가도 임시저장 유지. Step 번호 자동 갱신
  • 복원: 임시저장 항목 클릭 시 해당 Step + 대화 내용 그대로 복원
