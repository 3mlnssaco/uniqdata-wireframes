0. 페이지 역할

이 페이지가 답하는 질문: "이 연구, 지금 잘 되고 있나?"
- 참여자 유지율, 수집률, 예산 소진율, 알림 → 연구 운영 건강도 한눈에
- 군간 핵심 변화값은 "효과가 나타나고 있는가?"에 대한 빠른 확인
- 상세 데이터 분석은 테이블 페이지, AI 인사이트는 AI 허브 페이지에서

페이지 역할 분리 원칙:
- 개요 (이 페이지): 연구 건강도 — "잘 되고 있나?"
- 테이블: 환자별 상세 추적 — "각 환자 상태는?"
- AI 허브: AI 발견 인사이트 — "AI가 뭘 찾았나?"

1. Phase 기반 연구 설계

- Phase 1 (관찰기, W0-W2): 양군 모두 무투약, baseline 데이터만 수집
- Phase 2 (투약기, W3-W12): 실험군=위고비 2.4mg, 대조군=위약
- 모든 Δ값은 Phase 1 종료 시점(W2) 대비로 계산
- 연구자가 Phase 기간/수를 자유롭게 설정 가능해야 함

Phase 유연성:
- 기본: 2-Phase (관찰 → 투약)
- 확장: N-Phase (관찰 → 투약A → 워시아웃 → 투약B 등 crossover)
- Phase별로 측정 항목과 빈도를 다르게 설정 가능

2. 수집·예산 통합 설계

- 수집 현황과 예산 현황은 같은 높이의 2칼럼으로 배치
- 둘 다 "연구 운영"에 속하는 정보이므로 하나의 섹션으로 그룹핑
- 디바이스별 수집률로 어떤 장비에서 누락이 많은지 파악
- 예산은 군별 분리 표시 + 소진 예측

3. 군간 변화값 테이블

- 그래프 대신 숫자 테이블로 핵심 비교
- 연구자에게 필요한 정보: Δ값, SD, 군간 차이, p값
- 시각적 추이가 필요하면 테이블 페이지에서 개별 환자로 확인
