📋 디자이너 노트: 회원가입 Step 1

페이지 목적: 개인/기관 계정 유형 선택. 첫인상이 중요!

사용자 행동:
1. 랜딩 페이지 → "시작하기" 클릭 → 이 페이지
2. 개인(대학원생, 의료진) vs 기관(병원, 제약사) 선택
3. "다음" 클릭 → Step 2로 이동

디자인 포인트:
• 7단계 진행 표시 (왼쪽 패널)
• 선택된 카드는 파란 테두리 + 배경색
• 기관 선택 시 → 팀 관리/권한 기능 활성화됨 (나중에)
• 라디오 버튼 스타일로 단일 선택 표현

⚙️ 기능 명세

핵심 기능:
• 계정 유형 선택 (개인 연구자 / 기관·기업)
• 선택 상태 로컬 저장 (sessionStorage)
• 유형별 기능 차이 안내 표시

비즈니스 로직:
• 개인: 개인 연구, 데이터 접근, AI 분석 기능
• 기관: 팀 관리, 권한 설정, 대규모 연구 기능 추가
• 선택 필수 (미선택 시 다음 버튼 비활성화)
