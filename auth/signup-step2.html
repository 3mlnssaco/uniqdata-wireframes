📋 디자이너 노트: 이메일 인증 (Step 2)

🟢 우선순위: Phase 1A (0~3개월)

회원가입 7단계 중 2번째. 이메일 소유권 확인 단계.

구성 요소
  • 이메일 입력 필드: 가입할 이메일 주소 입력란
  • 인증 코드 전송 버튼: 클릭 시 이메일로 6자리 코드 발송. 전송 후 "전송됨" 상태로 변경(초록)
  • 6자리 OTP 박스: 개별 박스 6개. 입력된 자리는 파란 테두리+배경. 미입력은 대시(-)
  • 재전송 링크: "코드를 받지 못하셨나요?" + 재전송 (60초 쿨다운)
좌측 패널

7-step 수직 진행 표시. 완료 단계=초록 체크, 현재=흰 원+굵은 라벨, 미진행=반투명. 모든 가입 step에서 동일 레이아웃 공유.

⚙️ 기능 명세

핵심 기능:
• 이메일 주소 입력 및 형식 검증
• 이메일 중복 확인 (기존 가입 여부)
• 6자리 인증 코드 발송 (5분 유효)
• 인증 코드 검증
• 재전송 기능 (60초 쿨다운)

비즈니스 로직:
• 인증 코드 5분 내 입력 필수
• 최대 5회 실패 시 10분 잠금
• 재전송은 60초 간격으로 제한
• 인증 완료 시 이메일 확정 저장
