# uniqdata-wireframes

## 역할

ERD, 와이어프레임, 리서치 문서, HTML 프로토타입 모음.

## 현재 상태 (2026-02-11)

참고용. 피봇 전 설계 자료이나, ERD 패턴과 와이어프레임 디자인은 Phase 1 개발 시 참고 가치 있음.

## 참고 가치

- ERD 패턴: 기존 `projects` 테이블에 `irb_number` 필드 존재 → IRB 확장 기반
- 와이어프레임: Enterprise Portal UI 참고
- 리서치 문서: 피봇 전 20개+ 리서치 (현재 `docs/STRATEGY.md`에 통합됨)

## 관련 문서

- 아키텍처: `../docs/ARCHITECTURE.md`
